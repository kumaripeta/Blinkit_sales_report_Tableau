Blinkit Sales Dashboard (Tableau)
Project Overview

This project is an interactive sales analysis dashboard built using Tableau for Blinkit – India’s last-minute delivery app.
The dashboard provides insights into sales performance, item distribution, outlet characteristics, and trends over time.

📊 Dashboard Preview

🎯 Objectives

Analyze total and average sales

Understand sales distribution by item type

Compare outlet performance across tiers and sizes

Identify trends based on outlet establishment year

Evaluate impact of fat content on sales

📈 Key Metrics

Total Sales: $1.20M

Average Sales: $0.14K

Number of Items: 9K

Average Rating: 4

📌 Key Insights
🥦 Sales by Item Type

Fruits & Vegetables and Snack Foods generate the highest sales (~$0.18M each)

Seafood and Breakfast contribute the least

🏪 Sales by Outlet Type

Supermarket Type1 has the highest sales (~$787K)

Grocery stores contribute moderate sales

🧈 Fat Content Analysis

Low-fat items dominate sales (~$776K)

Regular items contribute ~$425K

📍 Outlet Location Tier

Tier 3 locations generate the highest sales

Tier 1 has comparatively lower performance

📦 Outlet Size

Medium-sized outlets generate the most revenue (~$0.51M)

Small outlets contribute significantly but less than medium

📅 Sales Trend

Peak sales observed around 2018 (~$0.20M)

Stable growth observed after 2012
